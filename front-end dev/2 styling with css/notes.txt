CSS (Cascading Style Sheets) - Everything You Need to Know! 🎨
This guide covers CSS from absolute basics to advanced concepts. Let's make web pages look awesome!🚀

1. What is CSS? 🤔
CSS stands for Cascading Style Sheets.

HTML builds the structure (like a house's skeleton).

CSS adds style (paint, decorations, furniture).

2. Types of CSS (Ways to write CSS) ✍️
There are 3 ways to use CSS in HTML:

a) Inline CSS (Inside the tag) ❌ (Not Recommended)
HTML
<h1 style="color: blue; font-size: 20px;">This is an Inline CSS</h1>
b) Internal CSS (Inside the <head>) 🟡 (Okay for small single pages)
HTML
<head>
    <style>
        h1 { color: red; }
    </style>
</head>
c) External CSS (Separate .css file) ✅ (Highly Recommended)
HTML
<head>
    <link rel="stylesheet" href="style.css">
</head>
3. Basic CSS Selectors 🎯
How to target HTML elements to style them:

CSS
/* 1. Element Selector (Targets all specific tags) */
p {
    color: green;
}

/* 2. Class Selector (Targets elements with class="myBox") - Starts with a DOT (.) */
.myBox {
    background-color: yellow;
}

/* 3. ID Selector (Targets specific element id="mainTitle") - Starts with a HASH (#) */
#mainTitle {
    font-size: 50px;
}

/* 4. Universal Selector (Targets EVERY element on the page) */
* {
    margin: 0;
    padding: 0;
}
4. Colors & Backgrounds 🌈
CSS
body {
    background-color: #f4f4f4; /* Hex color code */
    color: rgb(255, 0, 0);      /* RGB color code */
}

.image-bg {
    background-image: url('my-image.jpg');
    background-size: cover;    /* Fit image perfectly */
}
5. The Box Model 📦 (VERY IMPORTANT)
Every element in CSS is a box. It consists of:

Content: The text/image itself.

Padding: Space inside the border.

Border: The line around the padding.

Margin: Space outside the border.

CSS
.box {
    width: 200px;
    height: 100px;
    padding: 20px;       /* Space inside */
    border: 2px solid black; /* The border line */
    margin: 30px;        /* Space outside */
}
6. Text Formatting 📝
CSS
.text-style {
    font-family: Arial, sans-serif; /* Change font style */
    font-size: 24px;                /* Text size */
    font-weight: bold;              /* Text thickness (bold, normal, 100-900) */
    text-align: center;             /* Align: left, right, center, justify */
    text-decoration: underline;     /* Line under, over, or through text */
    text-transform: uppercase;      /* uppercase, lowercase, capitalize */
}
7. Flexbox 🤸‍♂️ (Modern Layout System)
Used for aligning items inside a container easily.

CSS
.container {
    display: flex; /* Turn on flexbox */
    
    /* Horizontal alignment */
    justify-content: center; /* center, space-between, space-around, flex-start, flex-end */
    
    /* Vertical alignment */
    align-items: center; /* center, flex-start, flex-end, stretch */
    
    /* Change direction (row is default) */
    flex-direction: column; 
}
8. CSS Grid 🔲 (Advanced Layout)
Best for 2D layouts (rows AND columns).

CSS
.grid-container {
    display: grid;
    
    /* Make 3 columns, each 100px wide */
    grid-template-columns: 100px 100px 100px; 
    
    /* Space between grid items */
    gap: 10px; 
}
9. Positioning 📍
How elements are placed on the screen.

CSS
.normal {
    position: static; /* Default */
}

.relative-box {
    position: relative; /* Moved relative to its normal position */
    top: 10px;
}

.absolute-box {
    position: absolute; /* Moved relative to its closest 'relative' parent */
    top: 50px;
    left: 20px;
}

.fixed-box {
    position: fixed; /* Sticks to the screen even when scrolling (like navbars) */
    top: 0;
}
10. Hover Effects & Transitions ✨
Making things move smoothly when interacted with.

CSS
.btn {
    background-color: blue;
    color: white;
    /* Tells CSS to animate the change over 0.3 seconds */
    transition: all 0.3s ease; 
}

/* What happens when mouse is OVER the element */
.btn:hover {
    background-color: red;
    transform: scale(1.1); /* Makes it 10% bigger */
}
11. Media Queries 📱 (Responsive Design)
Making your website look good on mobile phones.

CSS
/* Default style (Desktop) */
body {
    background-color: lightblue;
}

/* If screen width is 600px or LESS (Mobile phones) */
@media screen and (max-width: 600px) {
    body {
        background-color: lightgreen; /* Change background for mobile */
    }
    .myBox {
        width: 100%; /* Make boxes full width on small screens */
    }
}
