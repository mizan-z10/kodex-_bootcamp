## 1. Dice Game Create two dice images dynamically using `document.createElement()`. Use `Math.random()` to generate numbers (1--6). Display both dice results and show the winner based on the higher number. Render everything dynamically using `append()` or `appendChild()`.

## 2. Random Color Card Generator Create a card on button click. Generate a random RGB background color using `Math.floor()` and `Math.random()`. Set a random width and height. Assign a unique `data-id` using `setAttribute()`.

## 3. Lottery System Take user input (1--50). Generate a random number using `Math.ceil()`. If the number matches user input, display "You Won" otherwise "Try Again". Update result dynamically in the DOM.
