1. What is Vite

Vite = Modern frontend build tool
Use: Fast web apps (React, Vue, Svelte)

👉 Created by Evan You

Why Vite is Popular
❌ Old Approach (Webpack)
Bundle everything → then run

👉 Slow startup

✅ Vite Approach
Browser requests module
        ↓
Vite serves that module
        ↓
Instant start
Advantages

⚡ Fast dev server

🔥 Instant HMR (Hot Reload)

📦 Lightweight

🚀 Optimized build

2. Create React App using Vite
Step 1 — Create Project
npm create vite@latest
Step 2 — Enter Folder
cd my-react-app
Step 3 — Install
npm install
Step 4 — Run Project
npm run dev

👉 Output:

http://localhost:5173

👉 Root component

vite.config.js
import { defineConfig } from "vite";
import react from "@vitejs/plugin-react";

export default defineConfig({
  plugins: [react()],
